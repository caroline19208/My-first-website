Feature tags

Web pages can include text, links, tables, images and even tables.

Paragraphs hold text. They are placed below headings. It's defined with the 'P' tag. There are automatically line breaks between paragraphs. However, html doesn't automatically have line breaks for content within paragraphs. This has to be done manually, with the 'br' tag.
The 'br' tag doesn't need an end tag.

Attributes provide additional information about HTML elements (anything that goes between a start and end tag).

Links are defined with the 'a' tag, which stands for 'anchor'. They use the href (hypertext reference) attribute, which specifies the URL of the page the link goes to.

Style tags

These aren't used that much anymore because now we have CSS, but they're still useful to know.

The 'b' and 'strong' tag both do the same thing, bolding the sentence.
The 'i' and 'em' tag also do the same thing, slanting the sentence.
The 'u' tag undelines the sentence.

Divisions

Divisions define a section or division within the HTML document. This makes it easier to apply styles to different types of tags by grouping them into sections.

List tags

The 'li' tag defines a list item. There are 2 main types of lists.

Ordered list:

  1. Apples
  2. Bananas
  3. Mangoes

Although the default order starts with 1, 2, 3 etc. You can add the value attrubute to change this in an ordered list. For example like this:

  1. Cabbage
  2. Broccoli
  3. Aubergine

Unordered lists:

  • Water
  • Milk
  • Fruit juice

Script tags

These are used to embed a client-side script (JavaScript) and use the 'script' tag.
